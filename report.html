Final Project – Football Analytics

Hello, this is Kaden Heigley’s final project for Football Analytics. The final project is a map of all the current Steelers players on the roster, plus the free agents and draft picks the team has picked up for the most recent draft/season of the National Football League (NFL).

The map displays the United States with markers for every player, free agent, and draft pick. Clicking a marker shows the person's name, position, and college. The marker demonstrates the college location.

WEB Pages / URLs & Purpose

  • Steelers roster: steelers.com/team/players-roster
  • Wikipedia season page: 2025 Pittsburgh Steelers season
  • Traveling to Campus / New Student Orientation: used to confirm campus addresses

Tools

  • Leaflet — map rendering
  • Python — data processing and scripts
  • Pandas — CSV and table handling
  • BeautifulSoup — scraping

Files / DB

CSV files: college_raw, colleges, combined_table, current_roster, draft_picks, free_agents

Database: combined_table.db with table combined_table

Columns (example)

  • name — player's name
  • position — position
  • college — college/university
  • college_address, city, state, zip — for geolocation
  • team_status — Player / Free Agent / Draft
  • lat, long — coordinates for mapping

Queries

SELECT name, position, college, college_address, city, state, team_status
FROM combined_table

How to Use the Webapp

Open the map page, use the search bar to filter by name/college/position. Click markers to view details. The app loads players.geojson generated from the database.
